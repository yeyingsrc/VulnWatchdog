## 💡 使用技巧

### 1. 快速搜索
使用 `grep` 快速搜索关键词：
```bash
# 在所有文档中搜索"更新检测"
grep -r "更新检测" *.md

# 搜索配置相关内容
grep -r "config\|配置" *.md
```

### 2. 文档对比
比较不同阶段的优化：
```bash
# 对比P1和P2的优化重点
diff OPTIMIZATION_SUMMARY.md OPTIMIZATION_PHASE2_COMPLETE.md
```

### 3. Markdown预览
- VS Code: 安装 Markdown Preview Enhanced 插件
- GitHub: 直接在仓库中查看
- 命令行: 使用 `pandoc` 转换为HTML
